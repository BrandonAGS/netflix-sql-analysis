# Netflix SQL Analysis

Proyecto de análisis de datos en SQL utilizando una base de datos simulada tipo Netflix.

## Objetivo
Analizar información de series, episodios y actores para responder preguntas de negocio mediante consultas SQL.

## Herramientas
- MySQL
- SQL

## Modelo de Datos
El proyecto utiliza un modelo relacional compuesto por:
- Series
- Episodios
- Actores
- Actuaciones

## Habilidades aplicadas
- Consultas SELECT y filtros
- JOINs (INNER JOIN)
- Agregaciones (COUNT, AVG)
- GROUP BY y HAVING
- Subconsultas
- Ordenamientos y LIMIT

## Ejemplos de análisis
- Series con mejor rating promedio
- Episodios mejor calificados
- Cantidad de episodios por temporada
- Participación de actores por serie

## Estructura del proyecto
- `schema.sql` → Creación de tablas
- `data.sql` → Inserción de datos
- `sql_basico.sql` → Consultas básicas con JOINs
- `sql_intermedio.sql` → Consultas intermedias con JOINs y subconsultas

## Notas
Algunas consultas pueden presentar empates en resultados; se muestra uno representativo para fines de análisis.
