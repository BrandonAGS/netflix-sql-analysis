# Netflix SQL Analysis

Proyecto de práctica en SQL utilizando una base de datos simulada tipo Netflix.

## Objetivo
Analizar información de series, episodios y actores para responder
preguntas de negocio usando SQL.

## Herramientas
- MySQL
- SQL

## Modelo de Datos
- Series
- Episodios
- Actores
- Actuaciones

## Consultas realizadas
- Listado y filtrado de series
- Análisis de ratings por serie y género
- Uso de JOINs (INNER JOIN)
- Agregaciones con COUNT, AVG
- GROUP BY y HAVING
- Ordenamientos y LIMIT

## Ejemplos de análisis
- Series con mejor rating promedio
- Episodios mejor calificados
- Cantidad de episodios por temporada
- Participación de actores por serie

## Notas
Algunas consultas presentan empates en los resultados.  
En esos casos se muestra uno de los valores para fines de análisis.
